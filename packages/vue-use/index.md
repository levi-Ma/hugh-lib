# vue-use

Vue Hook 函数集

## Installation

::: code-group

```sh [npm]
$ npm i @hugh-ma/vueuse
```

```sh [pnpm]
$ pnpm add @hugh-ma/vueuse
```

```sh [yarn]
$ yarn add @hugh-ma/vueuse
```
:::

## Menu Tree

```bash
@hugh-ma/vueuse
├─ useCheckUpdate # 微信小程序检查版本更新
└─ useUniStorage # 一个封装 uniStorage 的缓存方法
```
